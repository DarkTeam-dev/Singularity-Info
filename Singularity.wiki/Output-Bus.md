# Output Bus
Output Bus - блок, через который многоблочная машина отдаёт предметы. Один многоблок может иметь больше одного Output Bus.

По-умолчанию Output Bus просто копит предметы, которые следует забирать трубами, настроенными на забирание предметов из инвентаря. Отвёрткой можно сменить режим на автовыгрузку, и тогда Output Bus самостоятельно будет "выплёвывать" предметы в инвентарь, в который смотрит (данный режим автовыгрузки введён в ИИС с версии 1.16).

Количество слотов внутри блока зависит от тира:
| Тир | Слотов   |
|-----|----------|
| ULV | 1        |
| LV  | 4 (2x2)  |
| MV  | 9 (3x3)  |
| HV  | 16 (4x4) |
| EV  | 25 (5x5) |
| IV  | 36 (6x6) |
| LuV | 49 (7x7) |
| ZPM | 64 (8x8) |
| UV  | 64 (8x8) |
| MAX | 64 (8x8) |

При заполнении ячеек этой шины многоблок всё равно продолжит работу. Предметы, которые не могут быть помещены в Output Bus, сначала будут выкладываться в Output Hatch (ячейка для капсул), а затем просто пропадать.
