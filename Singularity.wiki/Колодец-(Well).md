# Колодец (Well).
Представляет собой примитивный блок без внутреннего интерфейса. Применяется для геологической разведки жидкостей GregTech, находящихся под коренной породой. По умолчанию выдает результат работы на два типа жидкости: **OIL** и **WATER**. Взаимодействие с блоком, после расположения его в мире, возможно посредством правой кнопкой мыши (ПКМ). Если нажимать ПКМ, то в **чат игроку** будет выдаваться информация о процессе поиска: `Currently is N ready` (где N - процентное соотношение заполненности колодца, причём, если колодец сразу заполнился на 100%, это означает, что значение жидкости в этом чанке больше 1000); или `Currently is empty` - в случае не обнаружения какой-либо жидкостей из списка возможных.

Список разрешенных жидкостей задается в конфигурационном файле **MachineStats.cfg** строковым параметром `S:well.allowedFluids_water_oil=water|oil`. В результате сканирования верхняя часть колодца приобретет цвет текстуры найденной жидкости, в случае **OIL** - черный, **WATER** - голубой. В этом случае возможно взаимодействовать с колодцем имея в руках **одно ведро** или **одну жидкостную капсулу**, тем самым вычерпав из него найденную жидкость, после чего запустится новый цикл наполнения колодца.

Снимается с помощью ключа (Wrench), как обычный механизм из мода.

![Пример взаимодействия с колодцем](https://i.imgur.com/r6nacY6.png)

На данном примере обозначены четыре зоны жидкостей коренной породы разным цветом. В синей зоне обнаружена нефть (OIL), красной — вода (WATER), желтой и зеленой — не обнаружено воды или нефти, возможно, в этих зонах находится иной тип жидкости.
