**Large Heat Exchanger** (LHE, Большой теплообменник) - многоблочная машина для преобразования дистилированной воды в пар за счёт передачи тепла от горячей жидкости без затрат электроэнергии. Горячей жидкостью могут быть только две жидкости: Lava или Hot Coolant.

LHE при активации имеет период нагрева около 2 минут для достижения полной эффективности, а после потери подачи горячего хладагента остывает около 40 секунд, чтобы вернуться к КПД 0%.

Преобразование в пар возможно только из дистилированной воды. Любая другая вода или жидкость, залитая в Input Hatch для дистилированной воды, приведёт к взрыву механизма. К взрыву приведёт и отсутствие дистиллированной воды. Количество потребляемой воды зависит от количества потребляемой горячей жидкости.

LHE может выдавать Steam или Superheated Steam. Типа пара зависит от скорости подачи горячей жидкости. При малой скорости получении горячей жидкости генерируется Steam, при превышении некоторой скорости получения горячей жидкости, которая называется порог перегретого пара, генерируется Superheated Steam. Порог перегретого пара различен для горячих жидкостей. Порог перегретого пара для **лавы** составляет **1000** л/с, для **Hot Coolant** - **4000** л/с.

**Рассчёт для случая, когда горячей жидкости меньше, чем порог перегретого пара (100% эффективность, без Integrated Circuit):**

|   | Lava | Hot Coolant |
| --- | :---: | :---: |
| Имеет на входе столько горячей жидкости, л/с | 800 | 3200 |
| Потребляет дистил. воды, л/с | 800 | 3200/2=1600 |
| Выдаёт пара, л/с | 800*160=128000 | 1600*160=256000 |
| Название пара | Steam | Steam |
| Выдаёт охлаждённую жидкость, л/с | 800 | 3200 |
| Название охлаждённой жидкости | Pahoehoe Lava | Coolant |

**Рассчёт для случая превышения порога перегретого пара:**

|   | Lava | Hot Coolant |
| --- | :---: | :---: |
| Имеет на входе столько горячей жидкости, л/с | 1000 | 4000 |
| Потребляет дистил. воды, л/с | 1000/2=500 | 4000/4=1000 |
| Выдаёт пара, л/с | 500*160=80000 | 1000*160=160000 |
| Название пара | Superheated Steam | Superheated Steam |
| Выдаёт охлаждённую жидкость, л/с | 1000 | 4000 |
| Название охлаждённой жидкости | Pahoehoe Lava | Coolant |

Стоит отметить, что поток горячей жидкости не обязательно регулировать, как это обычно принято для многоблочных турбин. Если в Input Hatch'е окажется горячей жидкости больше, чем порог перегретого пара, то автоматически заберётся в 2 раза больше жидкости, чем число порога. Для примера, если лавы во входном люке окажется 3000 л, то машина заберёт 2000 литров лавы. Конечно, при желании всегда можно регулировать входной поток.

Если внутрь контроллера поместить сконфигурированную Integrated Circuit (интегральная схема), это снизит порог, при котором большой теплообменник производит перегретый пар. Каждая конфигурация схемы, которая больше больше 1, снижает порог перегретого пара на 150 литров за счет потери 1,5% эффективности.

## Постройка

Основание 3x3. Высота 4 блока.

Минимально требуются следующие блоки:
* Контроллер располагается в нижнем ряду посередине любой из сторон.
* Stable Titanium Machine Casings - корпус, минимально 24 штуки.
* В центре, который обычно у некоторых других многоблоков, пустой - 2 блока Titanium Pipe Casing.
* 1 Input Hatch для лавы или для Hot Coolant - в центре основания многоблока.
* 1 Input Hatch для дистилированной воды - в любом месте.
* 1 Output Hatch для выдачи охлаждённой жидкости (Pahoehoe Lava или Coolant) - в центре верхней части многоблока.
* 1 Output Hatch для выдачи Steam или Superheated Steam - в любом месте.
* Maintenance Hatch - в любом месте.
