# Input Bus
Input Bus - блок, через который многоблочная машина получает предметы. Один многоблок может иметь больше одного Input Bus.<br>
Количество слотов внутри блока зависит от тира:
| Тир | Слотов   |
|-----|----------|
| ULV | 1        |
| LV  | 4 (2x2)  |
| MV  | 9 (3x3)  |
| HV  | 16 (4x4) |
| EV  | 25 (5x5) |
| IV  | 36 (6x6) |
| LuV | 49 (7x7) |
| ZPM | 64 (8x8) |
| UV  | 64 (8x8) |
| MAX | 64 (8x8) |
