Replicator (репликатор) - одноблоковая машина, позволяющая копировать предметы. Требует UU-matter и шаблон предмета для копирования, записанный в Data Orb.

UU-matter (материя) - жидкость, которая производится в механизме под названием [Mass Fabricator](https://github.com/IdealIndustrial/GT5Unofficial/wiki/Mass-Fabricator). Шаблон предмета для копирования должен быть предварительно записан в Data Orb в машине, называемой Scanner. Data Orb не расходуется.

Отсканированный Data Orb ![Data Orb](https://raw.githubusercontent.com/IdealIndustrial/IdealIndustrial.github.io/main/wiki/DataOrb.png) кладётся в обведённый слот:

![Replicator GUI](https://raw.githubusercontent.com/IdealIndustrial/IdealIndustrial.github.io/main/wiki/replicator-gui.png)

LV Replicator потребляет 32 еу/т. Количество метерии для копирование большинства материалов или элементов расчитывается исходя из атомной массы. Откуда выходит, что чем больше порядковый номер химического элемента в таблице Менделеева, тем больше требуется материи. Например, химический элемент индий (Indium) имеет атомную массу 114, из-за чего понадобится 114 литров материи.

Обычный репликатор не поддерживает копирование семян для огорода. Специально для семян существует похожий механизм Organic Replicator.
